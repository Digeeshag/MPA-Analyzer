# Marine Protected Area (MPA) Analyzer — Sri Lanka

## Overview
A geospatial analysis project mapping Sri Lanka's Marine Protected Areas (MPAs),
territorial sea zones, and Exclusive Economic Zone (EEZ) using GeoPandas.

## Project Goals
- Map Sri Lanka's 12 nautical mile territorial sea
- Map Sri Lanka's 200 nautical mile EEZ
- Load and visualize real MPA data from WDPA
- Identify which MPAs fall within which maritime zone
- Produce publication-ready static and interactive maps

## Tools & Libraries
- Python 3
- GeoPandas
- Matplotlib
- Folium
- Shapely

## Data Sources
- Natural Earth (naturalearthdata.com)
- GADM (gadm.org)
- WDPA — World Database on Protected Areas (protectedplanet.net)
